# Discrete_Structures_Notes_1

Logic Operation:
- Negation: 
- Conjunction (And): &&, ∧
- Disjunction (Or): ||, ∨

With && (If both are true):
- FF: False
- FT: False
- TF: False
- TT: True

With || (If one is true):
- FF: False
- FT: True
- TF: True
- TT True

Equivalent: two boolean formulas are equivalent if and only if they have the same truth values for all possibile inputs
PvQ = PuQ

Distributive Law:
P^(QvR) = (P^Q)v(P^R)
